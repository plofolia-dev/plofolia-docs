    Plofolia — プロジェクト概要¶

    関係者への紹介・説明用サマリ。詳細な技術仕様は各 docs/ ファイルを参照。

    命名由来¶

    Plofolia = Plot + Portfolia

    Plot（プロット）¶

    創作の途中段階、ラフ、WIP（Work In Progress）を投稿する SNS 的な側面を表します。 クリエイターが日々の制作過程をタイムラインに気軽に投稿し、フォロワーと交流する場です。

    Portfolio（ポートフォリオ）¶

    完成作品を整理して公開するポートフォリオとしての側面を表します。 シリーズ機能やプロフィールページを使って、代表作を閲覧者に見せるショーケースとして機能します。

    コンセプト¶

    クリエイターにとって SNS（Plot）とポートフォリオサイト（Portfolio）を一か所で管理できる プラットフォームを目指しています。 X や Instagram のように日々の WIP を投稿しながら、同時に完成作品をポートフォリオとして整理・公開できます。

    閲覧者にとっても、好みのクリエイターの制作過程を楽しみながら代表作にアクセスできる、 クリエイターとファンの距離が近い場所を提供します。

    プロダクトコンセプト¶

    Plofolia（プロフォリア） は、クリエイター向けの SNS プラットフォームです。 NSFW コンテンツおよび AI 生成コンテンツにも対応しており、既存プラットフォームで 不利な扱いを受けているクリエイターの受け皿となることを目指しています。

    X（旧 Twitter）などの既存プラットフォームでは、クリエイターは シャドウバン・不透明なアルゴリズム・突然の規制強化に常にさらされています。 Plofolia はクリエイターが自分の作品を安心して発表・収益化できる専用の場所を提供します。

    差別化ポイント（強み）¶

    1. シャドウバンがない — アルゴリズムの透明性を担保し、シャドウバン可視化ダッシュボードでエゴサーチ（自分への影響確認）が可能
    2. タイムコメント機能 — 動画・音声投稿に時刻付きコメントを付けられる（YouTube ライブ風オーバーレイ）。イラスト以外のメディアへのエンゲージメントを高める独自機能
    3. NSFW・AI アーティストに優しい — NSFW コンテンツや AI 生成作品を制限・差別しない方針。AI 学習拒否宣言機能も提供し、アーティストの権利を尊重
    4. 完全なタイムライン分離 — 手描き / AI 生成 / 映像 など制作スタイルごとにタブを分離し、ユーザーが見たいコンテンツだけ選べる

    ターゲット¶

    一次ターゲット（投稿者）¶

    • X でシャドウバンを受けたイラストレーター・映像作家・音声クリエイター
    • AI 生成コンテンツを発表する場を探しているクリエイター（AI 絵師・AI 映像作家）
    • NSFW 作品を規制なく発表したい漫画家・コスプレイヤー・写真家
    • 動画・音声作品のファンとリアルタイムに交流したいクリエイター

    二次ターゲット（閲覧者）¶

    • 好みのジャンル・国籍・スタイルでフィルタリングしてクリエイターを探したいユーザー
    • タイムコメントで推しの動画・音声作品に参加したいファン

    主要機能¶

    カテゴリ 機能
    タイムライン 手描き / AI 生成 / 映像の 3 タブ分離、地域フィルタリング
    投稿 画像（最大 4 枚）・動画・音声（カバー画像付き）対応
    クリエイター分析 シャドウバン可視化ダッシュボード
    タイムコメント 動画・音声に時刻付きコメント（YouTube ライブ風オーバーレイ）
    シリーズ投稿 複数作品をシリーズにまとめてプロフィールに表示
    多言語対応 日本語・英語・中国語・韓国語・スペイン語（5 言語）
    AI 学習拒否 設定画面からワンクリックで AI 学習拒否宣言・プロフィールにバッジ表示
    DM リアルタイムチャット（SSE）
    通知 リアルタイム通知（SSE）
    ミュート/ブロック ユーザー単位で非表示・ブロック、タイムラインから自動除外
    管理システム ユーザー BAN・投稿削除・監査ログ・バグ報告管理（Aurora 管理画面）
    招待コード制 クローズドベータ期間中は招待コード制で登録を制限（設定で切替可）

    技術構成（概要）¶

    層 技術
    フロントエンド Next.js 14 App Router + TypeScript + Tailwind CSS
    バックエンド Node.js + Express + TypeScript + Drizzle ORM
    データベース PostgreSQL 16
    インフラ AWS EC2（t3.micro / ap-northeast-1）+ Nginx + PM2
    認証 JWT（アクセス 15 分 + リフレッシュ 7 日）+ Google/X OAuth
    ストレージ ローカル（将来 S3 移行予定）
    CI/CD GitHub Actions + Slack 通知（#dev-alerts）
    コスト 現在 $20〜30/月。詳細は docs/ops/COST.md 参照

    ビジネスモデル¶

    現在¶

    • Ko-Fi による任意支援（手数料なし）

    将来計画（収益化フェーズ）¶

    • クリエイタープレミアム — 高度な分析・優先表示などの有料オプション
    • プラットフォーム手数料 — クリエイター間の有料コンテンツ販売に 15% 手数料（Stripe 経由）
    • 成人向け決済 — CCBill 連携（海外規制対応）
    • 損益分岐点・フェーズ別コスト予測: docs/mgmt/COST.md 参照
