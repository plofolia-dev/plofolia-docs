ソリューション

Plofolia = Plot + Portfolio

日々のWIP（制作途中）を気軽に投稿しながら、完成作品をポートフォリオとして整理・公開できる。 クリエイターとファンの距離が近いプラットフォーム。

命名: Plo(t) + (Port)folia 3 / 13
ターゲットユーザー

二種類のクリエイターを同時に取り込む

H

人間クリエイター

  • シャドウバンに悩むイラストレーター・漫画家
  • 映像・音声作品をファンと深く交流したい映像作家・ミュージシャン
  • WIPから完成作品まで一か所で管理したいクリエイター
  • AI普及で自作品の価値が低下していると感じている人
  • 表現の自由度が高い環境で発表したいクリエイター
A

AIクリエイター

  • AI作品を制限なく発表できる場所が欲しいクリエイター
  • 他プラットフォームでのBAN・差別リスクを避けたい
  • プロンプト試行・スタイル探求をWIPとして公開したい
  • AIコンテンツ好きのファンへ直接届けたい

AIクリエイターへの公平な扱いは他サービスにない差別化点。 OnlyFans のAI版に近いポジション。

5言語対応（日本語・英語・中国語・韓国語・スペイン語）で グローバルのクリエイターを最初から取り込める設計。

一次ターゲット: 最初の500人 — 日本のイラスト・映像・音声クリエイター 4 / 13
差別化機能

他サービスにない 4 つの独自機能

タイムコメント

動画・音声の再生位置に紐づくコメント機能。SoundCloud 的な音声エンゲージメントを SNS 上で実現。X・pixiv にはない独自体験。

SNS + ポートフォリオ一体設計

WIP投稿をポートフォリオ作品に紐づけるだけで、制作過程ツリー付きの 作品ページが完成。SNSとポートフォリオサイトを別々に管理する手間がゼロ。

AI学習拒否宣言

設定1クリックでプロフィールにバッジ表示 + noaiメタ情報付与。 WIP投稿の積み重ねが「人間が作った」制作過程の証跡にもなる。

ドメイン分離タイムライン

画像 / 映像 / 音声 の3タブ + 人間/AIフィルター + 人気タブ。 各ジャンルのファンへ確実に届く設計。クリエイターの投稿が埋もれない。

その他: DM・通知 SSE リアルタイム / シリーズ機能 / 5言語対応 / PWA（Web） / ネイティブアプリ化を検討 5 / 13

検討中 収益化方針は現在未定

決済手段・手数料率・対象ユーザー範囲のいずれも確定していない。

現在の月額固定費 $14 / 月

ユーザーが0人でも発生する固定費。EC2 + EBS + S3 + SES + ドメイン。

収益化を考える際の判断軸

  • クリエイターが価値を感じて対価を払える仕組みか
  • 扱うコンテンツの範囲に対応できる決済手段か
  • プラットフォーム手数料はクリエイターが納得できる水準か
  • 広告モデルはクリエイターへの不信感を招くため慎重に検討

今後の検討候補（確定ではない）

クリエイター間チップ / 月額サブスク / 単品販売 / プレミアム機能 / 法人向けプランなど、 複数のモデルが選択肢としてある。ユーザー獲得後に改めて議論する。

まとめ

なぜ今、Plofolia に投資するか

MVP は稼働済み

外部ユーザー招待前の本番稼働段階。60以上のAPIエンドポイント、 25以上のDBテーブル、5言語対応まで完成しており、 0からの構築コストが不要。

市場の追い風

X のシャドウバン問題・AI生成コンテンツの普及・クリエイター権利への関心の高まりが 重なり、クリエイター専用 SNS への需要が拡大している。 参入タイミングとして最適。

リスクが低い

インフラコスト $14/月から始められる。 収益化方針は未定だが、固定費が極めて低いため ユーザー検証に十分な時間をかけられる。

エンジニアチームへの提案

Next.js / Express / PostgreSQL / AWS という業界標準スタック。 コードベースは GitHub で管理済み・CI 完備。 副業エンジニアから正社員まで段階的に参加しやすい。
