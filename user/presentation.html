他サービスにない 4 つの独自機能

タイムコメント

動画・音声の再生位置に紐づくコメント（YouTube Live風オーバーレイ）。 動画・音声作品専用の深いエンゲージメント体験。X・pixiv にはない。

SNS + ポートフォリオ一体設計

WIP投稿をポートフォリオ作品に紐づけるだけで、制作過程ツリー付きの 作品ページが完成。SNSとポートフォリオサイトを別々に管理する手間がゼロ。

AI学習拒否宣言

設定1クリックでプロフィールにバッジ表示 + noaiメタ情報付与。 WIP投稿の積み重ねが「人間が作った」制作過程の証跡にもなる。

画像 / 映像 / 音声 の3タブ + 人間/AIフィルター + 人気タブ。 各ジャンルのファンへ確実に届く設計。クリエイターの投稿が埋もれない。

その他: DM・通知 SSE リアルタイム / シリーズ機能 / 5言語対応 / PWA対応 5 / 12

なぜ Stripe + CCBill の二本立てか？

Stripe は NSFW コンテンツへの利用規約が厳しく突然停止リスクあり。 CCBill は OnlyFans 等で実績のある成人コンテンツ専用決済。 手数料 15% は OnlyFans（20%）より低く、移住クリエイターへの訴求になる。
