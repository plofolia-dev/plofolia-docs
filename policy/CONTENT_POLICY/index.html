    Plofolia コンテンツポリシー¶

    運用モデレーションの判断基準。利用規約・ユーザー向けガイドラインの根拠となる内部文書。

    コンテンツ可否一覧¶

    カテゴリ 方針 備考
    実写コンテンツ ✅ OK R18は年齢確認必須（isAdult フラグ）
    3DCG・AI生成 ✅ OK AI生成は要表記（投稿時にAIフラグ必須）
    未成年に見える絵柄の性的描写 ❌ 絶対NG 下記参照
    グロテスク・ゴア ⚠️ 条件付きOK フィクション内はOK・NSFWフラグ必須。過度にリアルな描写はNG
    BDSM・SM表現 ✅ OK 合意ある成人間の描写・R18フラグ必須
    犯罪を促す内容 ❌ 絶対NG 即削除・永久BAN
    リベンジポルノ・非同意性的画像 ❌ 絶対NG 即削除・BAN・法執行機関通報
    AIディープフェイク（実在人物性的） ❌ 絶対NG 有名人・一般人問わず

    クリエイターの自己宣伝 ✅ OK サービスのコアユースケース
    外部サービスへのリンク ✅ OK スパム的な大量投稿はNG
    企業・ブランドアカウント ✅ OK クリエイターとして投稿する形式で

    未成年絵柄ポリシー¶

    性的描写は絶対NG・例外なし。

    根拠¶

    • 米国 PROTECT Act (2003): 実在しないキャラでも「未成年に見える性的描写」が連邦法違反になり得る
    • EU指令 2011/93/EU: 加盟国で同様の規制
    • 日本の都条例: 非実在青少年の性的描写を規制
    • 決済サービスの利用規約: CCBill・Stripeを含む全ての決済サービスが明示禁止。違反でアカウント停止 → 全クリエイターへの支払いが止まる
    • プラットフォームの存続リスク: 発覚した場合にサービス全体が停止・訴追されるリスクがある

    運用ルール¶

    • 年齢が明示されていないキャラクターは、投稿者が「18歳以上」と設定することを必須とする
    • 見た目が明らかに未成年なら成人設定でも NG（「成人設定です」の一文で免責されない）
    • 違反投稿は即削除・永久BAN（警告なし）
    • 繰り返し違反・悪質なケースは法執行機関へ通報

    リベンジポルノ・非同意性的画像ポリシー¶

    絶対NG。最重大カテゴリとして扱う。

    対象となるコンテンツ：

    • 同意なく撮影・共有された性的画像・動画
    • 元交際相手等の性的画像を無断で投稿する行為
    • AIで生成した実在人物の性的画像（ディープフェイク）

    対応：

    • 発見次第即削除・永久BAN
    • 被害者からの申告は最優先で対応（24時間以内）
    • 法執行機関への通報・情報提供に積極的に協力

    AIディープフェイクポリシー¶

    実在する人物（有名人・一般人を問わない）の顔や声を使用した性的コンテンツは絶対NG。

    • AI生成であっても実在人物の性的描写は名誉毀損・プライバシー侵害にあたる
    • 架空のキャラクター・オリジナルキャラのAI生成はOK（AI表記必須）

    二次創作ポリシー¶

    Pixiv・Fanboxと同等の方針（日本の同人文化を尊重）。

    ケース 方針
    成人設定の版権キャラのR18 ✅ OK
    公式設定で未成年のキャラのR18 ❌ NG（未成年絵柄ポリシーと同様）
    権利者からの削除要請（DMCA） 受領後72時間以内に対応
    AI生成の二次創作 ✅ OK（AI表記必須）

    DMCAセーフハーバー維持のための必須対応¶

    著作権法上のセーフハーバー（プロバイダー免責）を維持するために以下が必須：

    1. DMCA申告受付窓口をフッター・利用規約に明記（例: dmca@plofolia.com）
    2. 申告受領後 72時間以内 に対象コンテンツを削除
    3. 削除したことを投稿者に通知し、異議申し立て手順を提供
    4. 申告・対応記録を保存（訴訟対策）

    DMCA対応を怠ると、プラットフォーム自体が著作権侵害の共同責任を問われる可能性がある。

    ハラスメント・ヘイトスピーチポリシー¶

    ハラスメント¶

    特定の個人に対する執拗な攻撃・嫌がらせは禁止。

    NGの例:

    • 特定ユーザーへの繰り返しの中傷・攻撃的投稿
    • 他ユーザーに対してBANや嫌がらせを呼びかける投稿
    • プライベートな会話・DMの無断公開

    OKの例:

    • 作品・投稿内容への批評・批判（人格攻撃でないもの）
    • 公的な議論・意見の相違

    ヘイトスピーチ・差別表現¶

    属性（人種・民族・国籍・性別・性的指向・宗教・障害等）を理由とした侮辱・差別的表現は禁止。

    NGの例:

    • 特定の民族・国籍を侮辱する表現
    • 性的マイノリティへの差別的発言
    • 障害者を蔑む表現

    OKの例:

    • 差別問題についての議論・批評（差別を助長しない形式）
    • 社会・政治問題についての意見表明

    グロテスク・ゴアポリシー¶

    OK（NSFWフラグ必須）¶

    • フィクション（漫画・イラスト・CGI）内の暴力・流血描写
    • ホラー作品の怪物・死体描写
    • 戦争・歴史をテーマにした暴力描写

    NG¶

    • 実在の事件・事故の残虐映像・画像
    • 過度にリアルで扇情的な傷口・解体描写（フィクションでも）
    • 動物虐待映像

    政治的コンテンツポリシー¶

    OK¶

    • 政治的意見・社会問題についての表明
    • 特定政党・政治家への支持または批判（事実に基づくもの）
    • 選挙・政治についての情報共有・議論

    NG¶

    • 選挙に関する組織的な操作・虚偽情報の拡散
    • 特定候補への投票を金銭等で誘導する行為
    • フェイクニュース・意図的なデマの拡散

    モデレーション手順¶

    通報受付フロー¶

    ユーザーが通報
  → Aurora 管理画面に通報が届く
  → 管理者がコンテンツを確認
  → 24時間以内に判断
      ├─ 違反: 投稿削除 + 投稿者に通知 + 必要に応じてBAN
      ├─ 違反なし: 通報を棄却
      └─ 判断困難: 保留 + 追加確認

    BAN基準¶

    違反レベル 例 対応
    軽微（初回・誤認の可能性） NSFWフラグ忘れ・軽微なスパム 警告 + 該当投稿削除
    中程度（繰り返し・悪質） ハラスメント・ヘイトスピーチ 7日間停止
    重大 未成年コンテンツ・doxxing・リベンジポルノ 即時永久BAN
    法的問題 犯罪に関わる可能性がある内容 永久BAN + 法執行機関へ通報

    Appeals（異議申し立て）¶

    BANまたはコンテンツ削除を受けたユーザーは support@plofolia.com に異議申し立て可能。

    • 申し立て後5営業日以内に回答
    • 重大違反（未成年コンテンツ・リベンジポルノ等）に対する異議申し立ては受け付けない
