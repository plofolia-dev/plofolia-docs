    Plofolia モデレーターハンドブック¶

    モデレーターが Aurora 管理画面を使って日常業務を行うための実務マニュアル。 コンテンツポリシーの判断基準は CONTENT_POLICY.md を参照。

    Aurora へのアクセス¶

    本番環境の Aurora 管理画面は外部公開されていない。SSH ポートフォワード経由でのみアクセスする。

    # EC2 接続情報は管理者から別途取得すること
ssh -i <ec2_key> -L 3001:localhost:3001 ubuntu@<ec2_ip> -N
# → ブラウザで http://localhost:3001 にアクセス

    ログインはユーザー名（またはメールアドレス）＋パスワード。アカウントは admin が発行する。

    対応優先度と目標時間（SLA）¶

    優先度 カテゴリ 目標対応時間
    P0（緊急） 未成年コンテンツ・リベンジポルノ・doxxing・犯罪予告 発見から 1時間以内
    P1（高） ハラスメント・スパム多発・なりすまし 24時間以内
    P2（中） NSFWフラグ漏れ・一般的な規約違反 72時間以内
    P3（低） 軽微な違反・確認が必要なグレーゾーン 7日以内

    P0 案件は夜間・休日でも即時対応。 判断に迷う場合は先に投稿を非公開化して確認時間を作る（後から復元可能）。

    日常業務フロー¶

    1. 通報キューの確認（毎日）¶

    1. Aurora → 「投稿報告」メニューを開く
    2. ステータス open でフィルタリング
    3. 古い順（created_at 昇順）に処理する
    4. カテゴリバッジで P0 案件（underage_content）を最優先で確認

    2. 投稿の判断¶

    通報を確認
  ├─ 明らかに違反 → 「投稿削除」+ ユーザーへの対応を検討
  ├─ 明らかに問題なし → 通報を `dismissed` に変更
  └─ グレーゾーン → 管理者メモに理由を記録して保留 or 管理者に相談

    3. 処理後の記録¶

    判断した根拠を 必ず管理者メモに残す。後から別のモデレーターが確認できるようにする。

    記録例:

    • 違反なし: 成人キャラの性的描写。NSFWフラグあり。問題なし。
    • 削除: 未成年に見える絵柄の性的描写。BAN 検討中。
    • 保留: グレーゾーン。管理者へエスカレーション。

    違反判断の基準¶

    未成年コンテンツ（最重大・P0）¶

    以下のいずれかに該当する場合は即削除・永久BAN:

    • 見た目が明らかに未成年なキャラクターの性的描写（投稿者が「成人設定」と書いても不可）
    • 身体的特徴（体格・顔立ち・制服等）から未成年と判断できる
    • 公式設定で未成年のキャラクター（年齢が明示されている場合）

    判断に迷う場合の原則: 疑わしければ NG。削除してからユーザーに確認する。

    ハラスメント¶

    NG:

    • 同一ユーザーへの繰り返しの攻撃的返信（3回以上）
    • 「○○をBANしろ」などの集団嫌がらせの呼びかけ
    • DM でのしつこい接触（被害報告がある場合）

    OK（判断に迷いやすいケース）:

    • 作品・投稿内容への批判・否定的レビュー → OK（人格攻撃でなければ）
    • 公開論争・口論 → 双方が当事者なら基本 OK

    スパム¶

    即削除:

    • 同一または酷似した内容を短時間に大量投稿（5件以上 / 1時間）
    • 無関係なハッシュタグを大量付与
    • フォロワー獲得目的の無差別コピー投稿

    グレーゾーン・判断が難しいケース¶

    ケース 対応方針
    ゴア表現がリアルすぎる NSFWフラグありでも削除検討。管理者に相談
    風刺・パロディの差別的表現 文脈で判断。意図が差別扇動なら NG
    二次創作で年齢不詳のキャラ 投稿者に成人設定の明示を求める。疑わしければ削除
    実在人物の批判的創作 性的でなければ OK。名誉毀損は別途判断

    BAN の操作手順¶

    一時停止・永久BAN¶

    1. Aurora → 「ユーザー管理」でユーザーを検索
    2. 「BAN」ボタンをクリック（admin 権限が必要）
    3. BAN 理由を入力（ユーザーには通知されない内部メモ）
    4. 永久 BAN の場合は「永久」にチェック

    注意: BAN は admin 権限のみ実行可能。moderator は BAN できない。該当ケースは admin にエスカレーションする。

    BAN レベルの目安¶

    違反 対応
    初回の軽微な違反（NSFWフラグ忘れ等） 警告のみ（投稿削除 + 管理者メモ）
    繰り返し違反・ハラスメント 7日間停止
    重大違反（未成年コンテンツ・doxxing・リベンジポルノ） 即時永久BAN
    犯罪に関わる可能性 永久BAN ＋ 法執行機関通報（admin 判断）

    ユーザーへの通知¶

    BAN 時のユーザー通知は現状自動送信されない。重大違反以外で初回 BAN の場合は、お問い合わせ経由でメッセージを送ることを検討する。

    異議申し立て（Appeals）の処理¶

    1. Aurora → 「BAN異議申し立て」メニューを確認
    2. ユーザーが申し立て理由を記載している
    3. 元の通報・削除理由を照合する
    4. 5営業日以内に判断・返答する

    却下すべきケース（異議を受け付けない）:

    • 未成年コンテンツ・リベンジポルノ・doxxing
    • 複数回の重大違反

    再検討すべきケース:

    • 誤認・別人の投稿だった可能性
    • ポリシーの解釈が曖昧で投稿者が善意だった可能性

    バグ報告・お問い合わせの振り分け¶

    Aurora で受け付けるバグ報告・お問い合わせの一次振り分け:

    内容 対応
    機能バグ・表示崩れ ステータスを resolved + 開発者に共有
    アカウント・ログイン問題 詳細をヒアリング。必要なら DB を直接確認
    コンテンツ削除への異議 「BAN異議申し立て」フローへ誘導
    支払い・課金（将来） 決済プロバイダーの問い合わせ先を案内
    法的な通知 即座に admin へエスカレーション

    エスカレーション基準¶

    以下は 必ず admin にエスカレーション する:

    • P0 案件（未成年コンテンツ等）で永久 BAN が必要
    • 法執行機関への通報が必要な可能性
    • メディアや弁護士からの連絡
    • サービス全体に影響する問題（スパム大量発生・DDoS 等）
    • BAN 判断に強い迷いがある場合

    よくある操作リファレンス¶

    投稿を削除する
  Aurora → ユーザー管理 → 投稿一覧 → 削除ボタン
  または: 投稿報告一覧 → 「投稿を表示」→ 削除

通報をクローズする（違反なし）
  投稿報告一覧 → 該当行を展開 → ステータスを「dismissed」に変更 → 保存

通報をクローズする（削除対応済み）
  投稿削除後 → ステータスを「reviewed」に変更 + 管理者メモに対応内容を記載

BAN 済みユーザーの確認
  ユーザー管理 → 「BAN済み」フィルタ

お問い合わせに返信する
  お問い合わせ一覧 → スレッドを展開 → 返信欄に入力 → 送信
  → ステータスが自動で「responded」に変わる
