    Plofolia デザイン方針¶

    フロントエンド実装時の UI デザインガイドライン。terra / aurora 両方に適用する。

    テーマ: Solarized Dark¶

    全 UI を Solarized Dark パレット（sol.* カラー）で統一する。 bg-black・text-white・text-gray-* の使用は禁止。

    カラーパレット¶

    Tailwind クラス Hex 用途
    sol-base03 #002b36 最も暗い背景・ボタン文字（濃い背景上）
    sol-base02 #073642 主背景
    sol-base01 #586e75 プレースホルダー・ラベル・補助テキスト
    sol-base00 #657b83 セカンダリテキスト
    sol-base0 #839496 本文
    sol-base1 #93a1a1 本文テキスト
    sol-base2 #eee8d5 見出し・ユーザー名（base3 の次に明るい）
    sol-base3 #fdf6e3 強調テキスト
    sol-yellow #b58900 ブックマーク・警告
    sol-orange #cb4b16 危険・破壊的操作
    sol-red #dc322f エラー・削除
    sol-magenta #d33682 (汎用アクセント)
    sol-violet #6c71c4 (汎用アクセント)
    sol-blue #268bd2 リンク・フォロー・未読バッジ
    sol-cyan #2aa198 主アクションボタン・タイムコメント
    sol-green #859900 成功・いいね

    テキストカラー規則¶

    text-sol-base3   → 強調テキスト（最明）
text-sol-base2   → 見出し・ユーザー名
text-sol-base1   → 本文
text-sol-base0   → セカンダリテキスト
text-sol-base01  → プレースホルダー・ラベル・補助テキスト
text-sol-base03  → ボタン文字（sol-cyan など濃い背景の上）

    アクセントカラーは text-sol-cyan・text-sol-blue・text-sol-yellow 等を目的に応じて使用する。

    背景・ボーダー¶

    • 主背景: bg-sol-base02（#073642）
    • カード背景: bg-sol-base02 または bg-sol-base03
    • ボーダー: border-sol-base01/30 〜 border-sol-base01/60（透明度付き）
    • bg-black は使用禁止。最も暗くしたい場合は bg-sol-base03（#002b36）を使う

    コンポーネント別ガイドライン¶

    PostCard¶

    • ボーダーレスフィード（カード間の区切りは色差のみ）
    • 各投稿の上端にカテゴリカラーアクセントバー（3px、border-t-[3px]）
    • アクションボタン: hover:bg-sol-cyan/10 hover:text-sol-cyan など hover 時にアクセントカラーを使用
    • タイムコメント数表示: Clock アイコン + カウント、0件は非表示

    VideoPlayer¶

    • カスタムコントロール（Solarized Dark テーマ）: bg-sol-base03、グラデーションオーバーレイ from-sol-base03/90
    • コントロール: 再生/一時停止・シークバー・音量（ミュートトグル + スライダー）・時刻・フルスクリーン
    • 操作後 2.5 秒でコントロール自動非表示
    • 複数プレイヤーの排他制御: usePlaybackStore

    AudioPlayer¶

    • カバーアート + 波形ビジュアライザー（AnalyserNode + Canvas）
    • 音量スライダー: w-20 accent-[#2aa198]（sol-cyan）
    • 時刻表示: 現在時刻 / 総時間（YouTube スタイル）
    • chatPanel prop でタイムコメントパネルをプレイヤー下部に表示

    フォーム・入力欄¶

    • 入力欄背景: bg-white/8 〜 bg-sol-base03/60
    • ボーダー: border border-white/15
    • フォーカス時: focus:border-sol-cyan/60
    • 送信・主アクションボタン: bg-sol-cyan・text-sol-base03

    モーダル・ドロップダウン¶

    • 背景: bg-sol-base02 または bg-sol-base03
    • オーバーレイ: bg-black/60
    • ボーダー: border border-sol-base01/40

    レイアウト原則¶

    高さ制限¶

    • タイムライン投稿: max-h-480px、object-cover（X 風クロップ: 16:9 / 4:5 / 1:1）
    • タイムコメントパネル: h-72 固定（h-auto は禁止。コメント量で枠が伸びる原因になる）
    • 動画は常に videoIsFullWidth=true で全幅表示（動画と画像の混在はない）

    タイムコメントビュー¶

    • 動画: VideoPlayer の chatPanel prop に <TimedComments /> を渡し、オーバーレイ表示
    • 音声: AudioPlayer の chatPanel prop に <TimedComments /> を渡し、プレイヤー下表示
    • hasVideoMedia フラグで動画・音声を判別し、重複して両方に渡さない

    禁止事項まとめ¶

    禁止 代替
    bg-black bg-sol-base03
    text-white text-sol-base3
    text-gray-* text-sol-base0 / text-sol-base01 など
    タイムコメントパネルに h-auto h-72 等の固定高さ
    絵文字をコードやドキュメントに含める テキスト表記を使用
